Thoughts

Crypto/Web3 community is where I operate. I stay up to date on all the latest news, be it SOL meme coins, EIP like ETH's account abstraction(7702), BTC core contributor discussions, Cosmos SDK bug bountys(igniteCLI, Tendermint). What i really love about blockchain is the underlying node infrastructure. I remember 2021, first leanring about how Tendermint reaches consensus, using Byzantine Fault tolerance mechanism. No one validator can corrupt a network, as they always need atleast 2/3 to validate the next block. In my mind, this is an art. Designing nodes, that sync data and construct blocks, filled with data, hashed, for cryptographic proof, and verification. All agreed on by an underlying concensus mechanism, that humans program, but once live is not easily tampered with. I can talk to you all day about PoW vs PoS(DPoS) vs PoA, and which I think is best.
